# 📊 Exploratory Data Analysis (EDA)

This project performs Exploratory Data Analysis (EDA) on an e-commerce sales dataset using Python. The analysis includes data cleaning, descriptive statistics, data visualization, and insight generation to understand customer behavior and sales trends.

## 🛠️ Tools Used
- Python
- Pandas
- NumPy
- Matplotlib
- Seaborn

## 📈 Key Highlights
- Data Cleaning & Preprocessing
- Descriptive Statistics
- Trend Analysis
- Outlier Detection
- Data Visualization

## 🚀 Conclusion
The project uncovers valuable sales patterns and customer preferences, helping support data-driven business decisions.
